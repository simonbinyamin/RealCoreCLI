### 1. Start Real Core CLI

```
ctrl + i
This will install and run the library.
```

##### dotnet run commands

```
ctrl + r  ->  dotnet run

ctrl + p  -> dotnet run --urls=https://localhost:44306

ctrl + w  ->  dotnet watch run
```
